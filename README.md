# BBJ Fullstack System (Laravel)

Sistem fullstack utama untuk **Berbagi Bites Jogja (BBJ)** yang dibangun menggunakan Laravel. Repository ini berfungsi sebagai sistem manajemen data donasi, relawan, dan operasional inti komunitas.

## Deskripsi
Proyek ini adalah repositori default untuk sistem manajemen internal BBJ. Menangani fitur-fitur utama seperti manajemen inventaris makanan yang diselamatkan, pendaftaran relawan, dan pelaporan dampak.

## Fitur Utama
- Manajemen Donasi Pangan
- Database Relawan & Tugas
- Dashboard Admin Operasional
- Sistem Pelaporan Dampak (Sustainability Metrics)
